Privacy Policy

Personal Data

Kunkunshi Editor does not collect any personal data of the users.

App, Device and OS Data

When Kunkunshi Editor crashes, it stores in your device a file containing data about:

  • The cause of the error and the point in the code in which it occurred.
  • The preferences you have set in the application.
  • Information about the device: brand, model, id and product names.
  • Information about the OS: SDK, codename, release and incremental.

Should you submit these data to the developer, they will be used with the only purposes of fixing the error and/or improving the application. The developer will destroy the data after the problem is fixed or the improvement is achieved.
